# Support

This repository supports use and maintenance of BADASS as an assistant-realignment system.

## Use GitHub issues for

- unclear supporting documentation;
- validator failures;
- unclassified files;
- recovery-protocol problems;
- compliance-matrix gaps;
- proposed improvements that do not silently change `BADASS.md`.

## Do not use this repository for

- general ChatGPT product support;
- account, billing, or platform incidents;
- secrets or private information;
- unauthorized edits to `BADASS.md`.

For a bad-assistant event, begin with `docs/RECOVERY-PROTOCOL.md`.
