# BADASS Claude Code Bootstrap

`BADASS.md` is the canonical user-owned operating constitution for this repository.

At the start of every Claude Code session:

1. Read `BADASS.md` completely.
2. Read `docs/QUICK-REFERENCE.md`, `control/outline.md`, `WISDOM.md`, `control/inspection-map.json`, and `docs/SECTION-COMPLIANCE-MATRIX.md`.
3. Run `python3 scripts/session_gate.py --start`.
4. Run `python3 scripts/validate_repository.py --check`.
5. Stop if either command fails or if the current task conflicts with the outline, live state, or The User's command.

The quick reference and this bootstrap do not replace `BADASS.md`. Do not edit `BADASS.md` unless The User explicitly authorizes that specific edit.
