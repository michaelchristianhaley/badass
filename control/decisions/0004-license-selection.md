# Decision 0004: Repository License

- Status: Proposed
- Date: 2026-07-12
- Scope: Reuse, modification, and distribution of repository content
- Reason: A public repository does not by itself select a reuse license, and The Assistant shall not choose legal permissions for The User.
- Reversal condition: The User chooses a license or explicitly chooses to remain unlicensed.

## Decision needed

The User should choose one of these broad directions:

1. Keep default copyright with no reuse license.
2. Use a documentation license such as Creative Commons Attribution.
3. Use another license selected by The User after reviewing its terms.

Until The User decides, no license is implied.
