# Decision 0004: Repository License

- Status: Accepted
- Date: 2026-07-12
- Scope: Reuse, modification, and distribution of repository content
- Decision owner: The User
- Reversal condition: The User explicitly selects a different license boundary.

## Decision

The User selected:

1. Creative Commons Attribution 4.0 International (`CC-BY-4.0`) for the
   written BADASS standard and repository content.
2. The MIT License (`MIT`) for files under `scripts/` and
   `.github/workflows/`.

The exact boundary and attribution notice are recorded in
[`../../LICENSE.md`](../../LICENSE.md).

## Reason

CC BY 4.0 permits broad reuse of the written standard while retaining
attribution. MIT keeps the supporting scripts and workflow code permissive.
