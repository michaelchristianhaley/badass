# Decision Records

Decision records preserve why the repository operates as it does.

## Statuses

- `Proposed`: awaiting The User's decision.
- `Accepted`: explicitly approved and active.
- `Superseded`: replaced by a newer accepted record.
- `Rejected`: considered and declined.

Accepted decisions are not silently rewritten. A changed decision receives a new record that links to the old record and marks it superseded.
