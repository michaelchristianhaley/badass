# Decision 0001: Repository Purpose

- Status: Accepted
- Date: 2026-07-12
- Scope: Entire repository
- Reason: The User explicitly defined the repository as a realignment source for an assistant after it has been a BAD ASSISTANT.
- Reversal condition: The User explicitly replaces this purpose.

## Decision

The repository shall prioritize assistant recovery, observable compliance, current-state verification, anti-Thrash discipline, anti-Drift discipline, and truthfulness.

`BADASS.md` is the governing source. Supporting files shall explain, test, and operationalize it without silently changing it.
